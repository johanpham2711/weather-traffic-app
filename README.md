# React + TypeScript + Vite - Weather Traffic App

###### by [Johan Pham](https://github.com/johanpham2711)

## 1. Install

```
git clone https://github.com/johanpham2711/weather-traffic-app.git
cd weather-traffic-app

# create .env file

# install dependencies
yarn
```

## 2. Run app

```
# run with dev mode
yarn dev
```
